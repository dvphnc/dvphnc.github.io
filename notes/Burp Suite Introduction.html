What is Burp Suite?

Think of it as a powerful magnifying glass and toolkit for web traffic. Used daily by pentesters, red teamers, and bug bounty hunters worldwide.

“If you can see it, you can change it.”

Proxy — The Middleman

Every request and response passes through Burp when you set your browser proxy to 127.0.0.1:8080.

GET /admin HTTP/1.1
Host: app.com
Cookie: session=abc123
User-Agent: Mozilla/5.0...

Repeater — Manual Testing Heaven

Send the same request 100 times while changing only one parameter.

POST /login HTTP/1.1
Content-Type: application/x-www-form-urlencoded

username=admin' OR '1'='1'-- -&password=x

Intruder — Automation Beast

Choose your attack type:

  • Sniper → fuzz one position
  • Battering Ram → same payload everywhere
  • Pitchfork → parallel lists
  • Cluster Bomb → every combination (dangerous!)

Sequencer — Token Entropy Check

Capture 5,000+ tokens → Burp tells you instantly if they’re predictable.

> 90 bits = strong

< 70 bits → easily guessable
